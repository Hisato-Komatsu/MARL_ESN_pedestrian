# MARL_ESN_pedestrian
This repository contains source codes of "Multi-agent reinforcement learning using echo-state network and its application to the pedestrian dynamics".

# Notes
-These codes require numpy, scipy, and Pillow.
-Most of the hyperparameter values are given in the class 'hyperparameters'. 
-Whether parameter sharing method is adopted or not is determined by the variable 'Env.experience_sharing'.
